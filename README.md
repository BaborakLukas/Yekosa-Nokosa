# KO:SA

**Team:** David MixIns, Popper Dell Cribt  
**Status:** Early development (concept + first mechanics)  
**Engine:** Godot Engine (GDScript)

## What is this game about?
KO:SA is an action/survival game about two brothers who collect **shards**, and fight off **bug-like enemies** to protect **The Mother**, a central entity keeping their world alive.
